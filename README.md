# RN_BBS
This is an app developmented by React Native & Redux for Android
# 演示
![](./BBS.gif)
# 优点
+ 全原生RN组件，无第三方组件（字体图标除外）
+ 使用Redux架构管理应用状态
+ 代码复用性强，如自封装公用组件HeaderView

# 缺点
+ 布局方面自适应做的比较一般
+ 首页下的各板块内容没有做按需加载，而是初始化时加载了所有内容
+ 功能上，如下载、收藏、转发等没有开发
